The Code

Three seeds. The cascade does the rest.

This is the entire configuration. Everything else is derived.

tokens/aether.css
/* Change these 3 values to rebrand everything */ :root { --seed-hue: 41; /* 0–360 → brand color */ --seed-sat: 72%; /* 0–100% → vividness */ --seed-lum: 50%; /* 0–100% → lightness */ /* 254 tokens derived automatically ↓ */ --color-primary: hsl(var(--seed-hue), var(--seed-sat), var(--seed-lum)); --color-primary-10: hsl(var(--seed-hue), var(--seed-sat), 93%); --color-accent: hsl(calc(var(--seed-hue) + 30), 85%, 55%); --fg: var(--color-neutral-70); --bg: var(--color-neutral-0); --border: var(--color-neutral-10); --ring-color: var(--color-primary-40); } /* Dark mode flips 30+ tokens automatically */ @media (prefers-color-scheme: dark) { :root { --fg: var(--color-neutral-80); --bg: var(--color-neutral-0); } }
Accessibility

Built in, not bolted on

Every component includes focus rings, reduced-motion support, and high-contrast mode. No opt-in required.

🎯
Focus Rings
19 :focus-visible rules across all interactive elements.
🛑
Reduced Motion
All durations zeroed to 0ms, easing curves linearized when prefers-reduced-motion is active.
🔲
High Contrast
Borders strengthen and focus rings widen when prefers-contrast: more is active.
