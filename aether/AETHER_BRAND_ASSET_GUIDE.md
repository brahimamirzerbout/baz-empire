## 4. VISUAL IDENTITY

### 4.1 Logo Concept

The ÆTHER logo embodies the **invisible layer** concept — ether as the medium through which all things connect and move.

**Primary Mark:** The Æ ligature (Æ) styled as:
- Two overlapping circles (connection/medium)
- A subtle scan-line or wave pattern through the letterform (digital transmission)
- The crossbar of Æ doubles as a QR code scan line (product tie-in)

**Logo Variants:**
| Variant | Usage | Format |
|---------|-------|--------|
| Full (ÆTHER + wordmark) | Website header, proposals, signatures | Horizontal |
| Icon (Æ monogram) | Favicon, social avatars, app icon | Square |
| Stacked | Business cards, vertical layouts | Vertical |
| Monochrome | Embroidery, stamps, low-fi prints | Single color |
| Reversed | Dark backgrounds | White/light |

### 4.3 Typography

#### Primary Font
**Inter** (by Rasmus Andersson)
- Weights: 400 (body), 500 (labels), 600 (buttons), 700 (subheadings), 800 (headings), 900 (display)
- Why: Modern, geometric, excellent at all sizes, free, variable font support

#### Arabic Font
**Noto Sans Arabic** (by Google)
- Weights: 400, 500, 600, 700, 800
- Why: Matches Inter's x-height and proportions, excellent Arabic legibility

#### Display Font (Optional)
**Space Grotesk** — for occasional display/headline use when Inter feels too corporate

#### Type Scale
| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| Hero H1 | 3.5rem (56px) | 800 | 1.1 |
| Section H2 | 2.5rem (40px) | 700 | 1.2 |
| Card H3 | 1.25rem (20px) | 600 | 1.3 |
| Body | 1rem (16px) | 400 | 1.6 |
| Small/Caption | 0.875rem (14px) | 500 | 1.5 |
| Overline | 0.75rem (12px) | 600 | 1.4 |
| Price | 3rem (48px) | 800 | 1 |

#### Type Rules
- **Never use ALL CAPS** for more than 3 words (overline/label excepted)
- **Maximum 2 font families** on any single page
- **Letter-spacing:** Headings -0.03em, Body 0, Overline +0.08em
- **Bilingual text:** FR first, AR toggle (never side-by-side paragraphs)

### 4.4 Iconography

**Style:** Outlined, 2px stroke, rounded caps/joins  
**Size:** 24×24px default, 16×16px small, 32×32px large  
**Source:** Lucide Icons (or Phosphor Icons for extended set)

**Custom Brand Icons (SVG):**
- 🇺🇸 US Tech badge → Custom SVG shield with American flag colors
- 📱 QR → Custom QR code scan animation icon
- 🎯 Intel → Eye/radar icon
- ⚡ Speed → Lightning bolt
- 👨‍💼 Direct → Person with signal icon

**Emoji Rules:**
- Use native emoji sparingly — only in social media, never in proposals or reports
- Brand materials use custom SVG icons only
- Exception: Pricing tables and feature lists may use ✅ and ❌

### 4.5 Photography & Imagery

**Style:** Dark, moody, tech-forward
- Deep purple/blue backgrounds
- Subtle gradient overlays
- Geometric patterns (grid lines, data visualization)
- No stock photos of people pointing at laptops
- No cliché handshake images
- QR codes as design elements (scattered, animated, as backgrounds)

**Image Treatment:**
- All photos: Darken 40%, apply violet gradient overlay at 15% opacity
- Product screenshots: Rounded corners (12px), subtle shadow, on dark background
- QR codes: Always shown in context (on table, on card, on phone)

## 6. BILINGUAL RULES

### Language Priority
1. **French** — Primary (business language in Algeria)
2. **Arabic (Darija)** — Secondary (B2C, social media)
3. **English** — Tertiary (international, tech audience)

### Implementation
- Toggle button: `FR | عربي` (top-right, always visible)
- Toggle switches all `data-fr` / `data-ar` attributes
- **Never mix FR and AR in the same paragraph** — toggle entire content blocks
- RTL support: `dir="rtl"` on `<html>` when Arabic active
- Font switch: Inter → Noto Sans Arabic when Arabic active

### Price Format
- **Always DA** (never DZD in copy — DA is what Algerians say)
- Format: `5 000 DA` (space as thousands separator)
- Per month: `5 000 DA/mois`
- One-time: `5 000 DA` (no period qualifier)

## 7. WRITING STYLE

### Tone Words
✅ Confident, Direct, Data-driven, Fast, Transparent, Bilingual  
❌ Arrogant, Vague, Corporate, Slow, Secretive, French-only

### Copy Rules
1. **Lead with data:** "27 concurrents analysés" not "nous avons beaucoup d'expérience"
2. **Show, don't tell:** "0/27 offrent des QR landing pages" not "nous sommes les seuls"
3. **Speed first:** "24h" not "rapidement"
4. **Prices visible:** Always show prices. Never "contactez-nous pour un devis"
5. **US tech = trust signal:** Reference React, TypeScript, Node.js, Netflix stack
6. **Bilingual respect:** Always provide FR + AR. Never FR-only in a bilingual market.
7. **Short sentences:** Max 20 words per sentence in French. Shorter in Arabic.
8. **One idea per paragraph:** If you have two ideas, you need two paragraphs.

### Forbidden Language
| ❌ Never | ✅ Always |
|----------|-----------|
| "Contactez-nous pour un devis" | "À partir de 5 000 DA" |
| "Solutions innovantes" | "Pages QR landing en 24h" |
| "Experts en marketing digital" | "27 concurrents analysés" |
| "Meilleur agence" | "Seule agence avec prix affichés" |
| "Synergie" | "Résultats" |
| "Accompagnement" | "24h chrono" |
| "Offre complète" | "Pack Complet à 30 000 DA/mois" |

## 12. BRAND GUARDRAILS

### Do ✅
- Use the Æ ligature wherever typographically possible
- Lead with data and competitive intelligence
- Show prices transparently
- Use dark mode as default (Void background)
- Reference "US tech" or "tech américaine" as a trust signal
- Provide FR + AR on all client-facing materials
- Use the Aether Gradient for primary CTAs
- Maintain generous whitespace
- Use Inter + Noto Sans Arabic exclusively

### Don't ❌
- Don't use the name "Zerbout Digital" anymore — fully transitioned to ÆTHER
- Don't use light theme as default (dark is our identity)
- Don't use generic stock photos
- Don't hide prices behind "contact us"
- Don't write French-only copy in the Algerian market
- Don't use more than 2 font families
- Don't use the gradient as body text color (only headings/highlights)
- Don't stretch or distort the logo
- Don't place the logo on busy backgrounds without a container
- Don't use Comic Sans, Papyrus, or any decorative font ever
