# BAZventures — Organizational Chart & Service Catalog

> Founder-led, AI-native agency. Triadic collective of senior-only specialists.
> Model contrasted against Ogilvy's matrixed, region × discipline structure.

## 1. Operating model

- **Shape:** Flat, founder-led collective — no regional hubs, no junior layer.
- **Core:** Three fused disciplines (Intelligence, Design, Execution) that mirror the Alpha → Beta → Gamma methodology.
- **Principle:** "No generalists. No junior staff. No learning curves." Every seat is senior and owns a function end-to-end.

## 5. Gaps vs. Ogilvy — recommended additions

| Function | Status | Why it matters |
|----------|--------|----------------|
| **Client Success / Account Director** | Missing | Ogilvy retains enterprise clients via dedicated account leads; you currently run accounts founder-direct. Add before scaling beyond ~5 active clients. |
| **Paid Media Buyer** | Partial (folded into Growth Operator) | Spend management is a distinct skill; split out once ad spend exceeds ~$20k/mo. |
| **PR & Influence** | Missing | Ogilvy's strongest vertical. Optional — only if you expand beyond technical/SMB into category-definition plays. |
| **Finance / Ops** | Missing | Invoicing, contracts, vendor accounts. Can stay founder-led early. |

## 6. Scaling path (grow without becoming Ogilvy)

1. **Now (founder + collective):** all 14 services, founder as integrator.
2. **Next (3–5 clients):** add **Client Success Lead**; promote one senior per discipline to "Head" (already mapped above).
3. **Later (scale):** spin **Paid Media** into its own seat; optionally add **PR** only if pursuing enterprise/category plays.
4. **Never copy:** Ogilvy's regional matrix — irrelevant below global-enterprise scale. Your edge is speed + technical depth, not headcount.
