# BAZ — Marketer & Agency Research Library

> Operator-grade research on the greatest marketers and agencies, distilled for BAZ.
> Compiled by the BAZ Agent. Sources: web research (Wikipedia, Britannica, agency sites,
> operator playbooks, industry reports). Tagged [fact] / [best-practice] / [assumption].

## Purpose

BAZ is a boutique, senior-partner-led marketing agency. This library catalogs the
foundational thinkers whose methodologies BAZ's operator lenses invoke, plus the agencies
whose work sets the ceiling. Each file: bio → signature work → core principles → BAZ
application.

## The BAZ Marketer Synthesis Sequence (default)

```
Godin (who/why) → Halbert/Schwartz (market & awareness) → Kennedy/Hormozi (offer)
→ Brunson/Deiss (journey) → Wiebe (words) → Ogilvy/Hopkins (test & prove)
```

Skipping market/offer lenses and jumping to copy = failure. No copy-first.

## How to use this library

1. Before any client engagement, run the synthesis sequence against the buyer's stage.
2. Pull the relevant marketer file for the lens you're invoking.
3. Every recommendation must improve one of: positioning, offer, lead flow, close rate,
   delivery quality, retention, scalability. If it improves none, cut it.
